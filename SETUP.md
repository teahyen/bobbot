변경사항이 반영되지 않으면:

- **Mac**: `Cmd + Shift + R`

2. Clear storage → Clear site data

## 🎯 사용 방법

### Step 1: 위치 선택
- **📍 현재 위치**: GPS 자동 감지
- **🗺️ 지도에서 선택**: 지도 클릭으로 원하는 위치 선택

### Step 2: 조건 선택
1. **음식 종류** (필수): 한식/중식/일식/양식
2. **음식 형태** (선택): 밥/빵/면
3. **거리** (필수): 10분/20분/30분

### Step 3: 검색
- **🔍 맛집 찾기** 버튼 클릭!
