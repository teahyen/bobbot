## 🔧 GitHub Pages 활성화 방법

### 1️⃣ GitHub 저장소로 이동
https://github.com/teahyen/bobbot

### 2️⃣ Settings 메뉴 클릭
- 저장소 상단의 **Settings** 탭 클릭

### 3️⃣ Pages 설정
1. 왼쪽 메뉴에서 **Pages** 클릭
2. **Source** 섹션에서:
   - Branch: **main** 선택
   - Folder: **/ (root)** 선택
3. **Save** 버튼 클릭

### 4️⃣ 배포 완료 대기
- 약 1-2분 후 페이지 상단에 배포 URL이 표시됩니다
- URL 형식: `https://teahyen.github.io/bobbot/`
