# 🍽️ 점매추 (점심 메뉴 추천 봇)

점심 시간, 무엇을 먹을지 고민되시나요? **점매추**가 여러분의 선호도와 위치를 기반으로 최적의 점심 메뉴를 추천해드립니다!

## 📋 주요 기능

### 1. 사용자 맞춤 선택
- **음식 종류**: 한식, 중식, 일식, 양식 중 선택
- **음식 형태**: 밥, 빵, 면 중 선택
- **거리 설정**: 걸어서 10분, 20분, 30분 이상 중 선택

### 2. 스마트 추천 시스템
- **실시간 위치 기반**: 현재 위치를 자동으로 감지
- **카카오맵 연동**: 정확한 주변 음식점 정보
- **거리순 정렬**: 가까운 곳부터 추천
- **지능형 필터링**: 선택한 조건에 맞는 맛집만 표시

### 3. 직관적인 UI
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- **인터랙티브 지도**: 클릭하면 해당 맛집으로 이동
- **상세 정보 제공**: 거리, 주소, 전화번호 표시

## 🚀 시작하기

### 1. 카카오맵 API 키 발급

1. [카카오 개발자 센터](https://developers.kakao.com/)에 접속
2. 내 애플리케이션 > 애플리케이션 추가하기
3. 앱 이름 입력 후 저장
4. 앱 설정 > 플랫폼 > Web 플랫폼 등록
5. 사이트 도메인 등록 (로컬 테스트: `http://localhost`)
6. 제품 설정 > 카카오맵 > Web 활성화
7. JavaScript 키 복사

### 2. API 키 설정

`index.html` 파일을 열고 다음 부분을 수정하세요:

```html
<!-- 기존 -->
<script type="text/javascript" src="//dapi.kakao.com/v2/maps/sdk.js?appkey=YOUR_KAKAO_API_KEY&libraries=services"></script>

<!-- 수정 후 -->
<script type="text/javascript" src="//dapi.kakao.com/v2/maps/sdk.js?appkey=발급받은_JavaScript_키&libraries=services"></script>
```

### 3. 프로젝트 실행

#### 방법 1: 간단한 HTTP 서버 (Python)
```bash
# Python 3가 설치되어 있다면
python3 -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

#### 방법 2: 간단한 HTTP 서버 (Node.js)
```bash
# npx가 설치되어 있다면
npx http-server -p 8000

# 브라우저에서 http://localhost:8000 접속
```

#### 방법 3: VS Code Live Server
1. VS Code에서 Live Server 확장 프로그램 설치
2. `index.html` 파일에서 우클릭
3. "Open with Live Server" 선택

## 💡 사용 방법

1. **위치 권한 허용**: 브라우저에서 위치 정보 접근 권한을 허용해주세요
2. **조건 선택**: 
   - 원하는 음식 종류 선택 (필수)
   - 음식 형태 선택 (선택사항)
   - 이동 가능 거리 선택 (필수)
3. **검색**: "🔍 맛집 찾기" 버튼 클릭
4. **결과 확인**: 
   - 지도에서 마커로 위치 확인
   - 아래 목록에서 상세 정보 확인
   - 클릭하면 지도가 해당 위치로 이동

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **API**: 카카오맵 API (Maps, Places)
- **Location**: Geolocation API

## 🔍 작동 원리

### 1. 위치 감지
- 브라우저의 Geolocation API를 사용하여 사용자의 현재 위치를 감지합니다
- 위치 정보를 가져올 수 없는 경우 서울 시청을 기본 위치로 설정합니다

### 2. 카테고리 매핑
```javascript
const categoryMapping = {
    '한식': ['한식'],
    '중식': ['중식', '중국집', '중국요리'],
    '일식': ['일식', '돈까스', '회', '초밥', '라멘'],
    '양식': ['양식', '이탈리안', '스테이크', '햄버거', '파스타']
};
```

### 3. 음식 형태 필터링
```javascript
const foodFormKeywords = {
    '밥': ['덮밥', '비빔밥', '볶음밥', '쌈밥', '정식', '백반', ...],
    '빵': ['베이커리', '샌드위치', '토스트', '햄버거', ...],
    '면': ['국수', '라면', '우동', '소바', '파스타', ...]
};
```

### 4. 검색 및 필터링 프로세스
1. 선택한 음식 종류의 모든 카테고리로 검색
2. 설정한 거리 내의 음식점만 조회
3. 중복 제거
4. 음식 형태로 추가 필터링 (선택된 경우)
5. 거리순으로 정렬
6. 결과 표시 및 지도에 마커 표시

## ⚠️ 주의사항

### API 키 보안
- **주의**: API 키가 코드에 노출되어 있습니다
- **운영 환경**: 서버 사이드에서 API 키를 관리하는 것을 권장합니다
- **도메인 제한**: 카카오 개발자 센터에서 허용된 도메인만 등록하세요

### 위치 권한
- HTTPS 환경에서만 정확한 위치 정보를 가져올 수 있습니다
- 로컬 개발 시 `localhost`는 예외적으로 허용됩니다

### 브라우저 호환성
- 최신 버전의 Chrome, Firefox, Safari, Edge 권장
- IE는 지원하지 않습니다

## 🔧 커스터마이징

### 검색 반경 변경
`index.html`에서 거리 옵션의 `data-value` 값을 변경하세요 (단위: 미터):
```html
<button class="option-btn" data-category="distance" data-value="500">걸어서 5분 이내</button>
```

### 음식 카테고리 추가
`app.js`의 `categoryMapping` 객체에 새로운 카테고리를 추가하세요:
```javascript
const categoryMapping = {
    '한식': ['한식'],
    '분식': ['분식', '떡볶이', '김밥'],  // 새로운 카테고리 추가
    // ...
};
```

### 음식 형태 키워드 수정
`app.js`의 `foodFormKeywords` 객체를 수정하여 더 정확한 필터링을 구현하세요:
```javascript
const foodFormKeywords = {
    '밥': ['덮밥', '비빔밥', '볶음밥', '추가키워드'],
    // ...
};
```
